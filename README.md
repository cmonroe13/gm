# gm
Guerrilla Mail CLI

```
usage: gm [<flags>] <command> [<args> ...]

Guerrilla Mail CLI.

Flags:
  --help     Show context-sensitive help (also try --help-long and --help-man).
  --version  Show application version.

Commands:
  help [<command>...]
    Show help.

  new
    Create new session.

  info
    Current session info.

  list
    List emails.

  sync
    Sync emails.

  read <id>
    Read email <id>.

  delete <id>
    Delete email <id>.
```
